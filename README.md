pass-generator
==============

A "strong" password generator written in Python

Generates ten 8-digit passwords that aren't a date in `ddmmyyyy` format

### Example  
```
python generator.py
```

*Output*:  

```
94120753  
18702563  
87304592  
41927380  
20813976  
58129063  
58219630  
93412578  
72340851  
29143086
```
